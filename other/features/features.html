Основные возможности Olimpus

Требования

Что из программного обеспечения необходимо на сервере для работы phpBB Olimpus? Какие типы баз данных поддерживаются и какая минимальная версия PHP необходима?


Поддерживаемые типы баз данных

Olimpus поддерживает различные типы баз данных. Если Вы - эксперт по одному из этих типов баз данных и работаете с этим ежедневно, компетентны в PHP, то у вас не возникнет потребности связаться с командой развития.

  • MySQL 3.23+/4.0+/4.1+/5.0+
  • MySQLi (MySQL 4.1+/5.0+)
  • PostgreSQL 7.3+
  • SQLite
  • Firebird 1.5+/Interbase 7.1+
  • Oracle
  • MSSQL 2000+
  • MSSQL 2000+ (ODBC)

Минимальная требуемая версия PHP

PHP >= 4.3.3



Основные возможности

Что улучшено?

Возможности, которые являются уже доступными в 2.0.x, были улучшены и/или расширены для Olimpus. Этот список содержит только самые важные усовершенствования.

Что является новым?

Возможности, которые не существовали в 2.0.x и были реализованы для Olimpus. Этот список не полон, он содержит только самые важные дополнения.

Личные сообщения

  • Массовая рассылка
  • Рассылка членам групп
  • Рассылка BCC (РАССЫЛКА ПЕРВЫХ ЭКЗЕМПЛЯРОВ)
  • Пересылка сообщения
  • Список друзей в качестве адресной книги
  • Прикрепление файлов к личным сообщениям
  • Черновик личного сообщения
  • Просмотр истории
  • Папки личных сообщений (почтовый ящик)
  • Фильтры для личных сообщений
  • Экспорт сообщений

Регистрация

  • Включение/отключение COPPA
  • Лимит попыток регистрации
  • Установка максимума\минимума длинны имени пользователя
  • Установка максимума\минимума длинны пароля
  • Установка ограничения символьного набора для имени пользователя
  • Установка требования сложности пароля
  • Возможность изменения пароля
  • Включение\отключение возможности смены адреса электронной почты

Поиск
